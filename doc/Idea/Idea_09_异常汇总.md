## 3.Address already in use: NET_Bind

### 3.1 异常信息

项目启动时，抛出如下错误信息：

![1565605841437](images/1565605841437.png)



### 3.2 异常原因

端口占用



### 3.2 异常解决

关闭占用的端口即可

```
netstat -ano | findstr 61185 //列出进程极其占用的端口，且包含 61185
tasklist | findstr 14592 // 查看是什么程序占用的
taskkill -PID 14592 -F //强制关闭某个进程
```
