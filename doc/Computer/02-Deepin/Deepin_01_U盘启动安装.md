# 二、设置启动顺序

> 安装基本没啥问题，不过我用一台电脑装完之后，重启老是进入window系统，通过如下方式可解决。

用过两台hp电脑，一台F10进入BIOS后，在设置管理员XXX，设置启动顺序后，按F10可以保存，将deepin顺序调成第一就可以开机进入deepin系统了。



但是另一台，设置启动顺序后，按F10可以无法保存，按什么键都没用。最终，再开机时按F9，进入开启启动项选择界面，选择deepin进入即可
